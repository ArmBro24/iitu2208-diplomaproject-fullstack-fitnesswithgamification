# **Diploma Project (Fullstack • Fitness With Gamification)**

Добро пожаловать в общий репозиторий дипломного проекта

Здесь будет вестись разработка **трёх модулей**:

* **Frontend**
* **Backend**
* **Database**

Каждый из них развивается **в отдельных ветках**, а коммуникация идёт через **отдельно развёрнутый API**

---

## **Цель проекта**

Создать full-stack систему **геймификации фитнес-процессов**, где пользователи смогут:

* выполнять тренировки;
* получать очки опыта и уровни;
* проходить челленджи;
* следить за прогрессом;
* взаимодействовать с тренерами;
* хранить данные в надёжной структуре

Проект демонстрирует навыки **бэкенда, фронтенда, проектирования, архитектуры и командной разработки**

---

## **Как мы работаем**

1. **Каждая задача = своя ветка.**
   Примеры:
   `feature/frontend-login-page`
   `feature/backend-auth-api`
   `bugfix/database-index-fix`

2. **Фронт, бэк и база — не смешиваются**
   У каждого направления своя зона ответственности

3. **Связка через API**
   Бэкенд поднимается отдельно -> фронт подключается по URL -> модульность

4. **Pull Requests обязательны**

   * проверка кода
   * согласование между направлениями
   * короткое описание изменений
